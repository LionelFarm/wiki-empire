# 🏅 Grades & permissions

Les grades sont gérés par **LuckPerms**. Ils déterminent ton préfixe dans le chat et tes avantages.

## Les grades

| Grade | Préfixe chat | Type |
|-------|-------------|------|
| **Joueur** | *(aucun)* | Grade de départ |
| **VIP** | `[VIP]` | Grade boutique / récompense |
| **Helper** | `[Helper]` | Staff — aide les joueurs |
| **Modérateur** | `[Moderateur]` | Staff — modération |
| **Admin** | *(aucun)* | Direction |
| **Owner** | *(aucun)* | Fondateur |

## Avantages par grade

### Nombre de bases (`/home`)

| Grade | Homes |
|-------|-------|
| Joueur | **1** |
| VIP | **3** |
| Staff | **illimité** |

Avec plusieurs homes, nomme-les : `/sethome <nom>`, puis `/home <nom>`.

{% hint style="info" %}
En plus du grade, tu as un **niveau de contribution** qui débloque des paliers chez les marchands. Voir [Niveau de contribution](../progression/contribution.md).
{% endhint %}

## Le chat

Le chat affiche ton préfixe de grade, ton éventuel tag de faction, puis ton pseudo :
```
[VIP] Steve : Salut tout le monde !
```
