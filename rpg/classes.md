# 🎭 Classes & compétences

EMPIRE Survival intègre un vrai système **RPG** via **MMOCore** : choisis une classe, monte en niveau, débloque des sorts.

## Choisir sa classe

```
/class
```
Ce menu te permet de **voir et choisir ta classe**. Les classes disponibles :

| Classe | Style |
|--------|-------|
| ⚔️ **Guerrier** | Mêlée, tank, résistance |
| 🔮 **Mage** | Sorts, dégâts magiques, mana |
| 🏹 **Tireur** | Distance, arcs, précision |
| 🛡️ **Paladin** | Soutien, soin, défense |
| 🗡️ **Roublard** | Vitesse, furtivité, dégâts critiques |

## Monter en niveau

Tu gagnes de l'**XP RPG** en jouant (combat, métiers…). Chaque niveau te donne des **points** à dépenser dans ta classe et ses compétences.

## Lancer des sorts

Le lancement de sorts se fait via la **barre de sorts** :

1. Appuie sur **F** (échange de main) pour ouvrir ta barre de sorts.
2. Appuie sur **1, 2, 3 ou 4** pour lancer le sort correspondant.

{% hint style="info" %}
Lancer un sort consomme de la **mana** (visible dans le HUD action-bar : ❤ PV / mana / armure). La mana se régénère avec le temps.
{% endhint %}

## Les métiers

MMOCore propose aussi des **métiers** qui montent en niveau quand tu les pratiques : minage, bûcheronnage, agriculture, pêche, fonderie, forge, alchimie, enchantement.

## Statistiques RPG

Tes caractéristiques (Force, Dextérité, Intelligence, Volonté…) augmentent tes dégâts, ta vie, ta mana, etc. Tu les améliores en montant de niveau et via ton [équipement RPG](equipement.md) et tes [reliques](reliques.md).
