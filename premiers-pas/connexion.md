# 🔐 Connexion & sécurité

EMPIRE Survival fonctionne en **offline mode** (comptes crackés autorisés). Pour protéger ton pseudo, le serveur utilise **TitanLogin** : à ta première arrivée, tu crées un mot de passe, puis tu te connectes avec à chaque session.

## Première connexion

À ton arrivée sur le serveur, tu seras gelé sur place jusqu'à t'être enregistré. Un message rouge s'affichera en permanence pour t'indiquer les commandes.

1. Tape `/register <motDePasse> <motDePasse>` pour créer ton compte
2. Ton compte est créé, tu peux jouer !

{% hint style="warning" %}
Ton mot de passe doit faire **au minimum 8 caractères**. Tu as **60 secondes** pour t'enregistrer, sinon tu seras expulsé.
{% endhint %}

## Reconnexion

Si tu te reconnectes dans les **2  heures** suivant ta déconnexion (depuis la même IP), tu seras automatiquement reconnecté sans retaper ton mot de passe.

Au-delà des **2 heures**, tape simplement :

```
/login <motDePasse>
```

## Gérer ton mot de passe

| Commande                             | Description              |
| ------------------------------------ | ------------------------ |
| `/changepassword <ancien> <nouveau>` | Changer ton mot de passe |

{% hint style="info" %}
Tu peux écrire dans le **chat** même avant de t'être connecté, pour demander de l'aide si tu as un problème.\
Si tu ne peux carrément pas te connecter, rejoins notre **discord** et fait un ticket, il sera très rapidement pris en charge.
{% endhint %}
