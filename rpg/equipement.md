# 🎒 Équipement RPG

En plus de ton armure classique, tu disposes d'un **inventaire d'équipement RPG** avec des emplacements spéciaux, géré par **MMOInventory**.

## Ouvrir ton équipement

```
/equipement
```
Alias : `/eq`, `/equip`, `/inv`.

## Les emplacements

| Emplacement | Ce qu'on y met |
|-------------|----------------|
| 🪖 Casque / Plastron / Jambières / Bottes | Ton armure (synchronisée avec l'armure normale) |
| 🛡️ Main secondaire | Bouclier / totem |
| 💍 **Anneau** | Un anneau RPG (bonus passifs) |
| 🧶 **Ceinture** | Une ceinture RPG (bonus passifs) |
| 🔮 **Relique** | Ta relique évolutive |

## À quoi ça sert ?

Les objets d'**Anneau**, **Ceinture** et **Relique** donnent des **bonus passifs** (PV, armure, dégâts, vitesse, mana…) **tant qu'ils sont équipés dans le bon slot**.

{% hint style="warning" %}
Une relique posée dans ton inventaire normal ne fait **rien**. Elle doit être placée dans le slot **Relique** de `/equipement` pour appliquer ses bonus.
{% endhint %}

👉 Pour fabriquer et faire évoluer ta relique, voir [Reliques évolutives](reliques.md).
