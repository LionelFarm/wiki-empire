# 🟣 Panoplie Empirium

L'**Empirium** est le matériau légendaire d'EMPIRE Survival : une ressource violette plus puissante que la Néthérite, qui sert à forger la panoplie d'élite.

## Trouver de l'Empirium

Le **minerai d'Empirium** génère naturellement dans les profondeurs (entre la couche -64 et 16, comme les diamants mais plus rare).

En le minant, tu obtiens des **Lingots d'Empirium**.

{% hint style="info" %}
L'Empirium est **~10 % plus rare** que le diamant : c'est un matériau d'endgame, garde-le précieusement.
{% endhint %}

## La panoplie Empirium

Avec des Lingots d'Empirium, tu peux forger un set complet **+15 % par rapport à la Néthérite** :

| Objet | Bonus |
|-------|-------|
| ⚔️ Épée Empirium | ~9,2 dégâts |
| 🪓 Hache Empirium | ~11,5 dégâts |
| ⛏️ Pioche / Pelle / Houe Empirium | Outils renforcés |
| 🛡️ Bouclier Empirium | Défense accrue |
| 🪖 Armure Empirium (4 pièces) | +15 % armure, +ténacité, durabilité ~2800 |

## Bonus de set

Porter plusieurs pièces de la panoplie Empirium débloque des **bonus de set** progressifs (2, 3 puis 4 pièces équipées).

{% hint style="warning" %}
La panoplie Empirium se fabrique à l'**établi** avec des Lingots d'Empirium (recettes dédiées). Combine-la avec une [relique](reliques.md) et une [classe RPG](classes.md) pour devenir une vraie machine de guerre.
{% endhint %}
