# 🧭 Premiers réflexes

Tu viens d'arriver sur EMPIRE Survival ? Voici l'essentiel pour bien démarrer.



## 2. Mets ton skin

```
/skin <pseudo_mojang>
```

👉 Voir [Ton skin en jeu](skin.md).

## 3. Pose ta base

Quand tu as trouvé un bon endroit :

```
/sethome
```

Pour y revenir à tout moment : `/home`

## 4. Explore les commandes RPG

Ouvre ton profil RPG, tes compétences et ta classe :

```
/profile (tes métiers)
/skills (tes sorts)
/skilltrees (arbre de commpétences)
```

👉 Voir [Classes & compétences](../rpg/classes.md).

## 5. Ouvre ton inventaire RPG

Ton inventaire RPG contient des emplacements spéciaux (anneau, ceinture, relique) :

```
/eq
```

## 6. Le spawn

Pour rentrer au spawn à tout moment :

```
/spawn
```

{% hint style="warning" %}
⚔️ En **combat PvP**, tu es **tag combat pendant 30 secondes** : impossible de te téléporter (`/home`, `/spawn`, `/tpa`…)de te déconnecter sans mourir ni d'utiliser des élytres.
{% endhint %}
