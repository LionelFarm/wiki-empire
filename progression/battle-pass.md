# 🎟️ Battle Pass

Le **Battle Pass** récompense ta progression sur le serveur avec de l'argent, des clés de coffres et des lingots d'Empirium.

## Ouvrir le Battle Pass

```
/bp
```

## Comment ça marche ?

* Le Battle Pass compte **50 paliers**.
* Tu gagnes de l'**XP de pass** en jouant et en complétant des **quêtes**.
* À chaque palier atteint, tu **réclames** tes récompenses dans le menu `/bp`.

## Les pistes de récompenses

| Piste | Accès | Récompenses |
|-------|-------|-------------|
| 🟢 **Scout** | Gratuit (tout le monde) | Argent + clés communes régulières |
| 🔵 **Vétéran** | Premium | Plus d'argent, clés rares, lingots Empirium |
| 👑 **Empereur** ☠ | Premium élite | Le maximum : clés légendaires, gros lots de lingots |

## Types de récompenses

* 💰 **Argent** ($) — montant croissant à chaque palier
* 🗝️ **Clés de coffres** — communes, rares et légendaires
* 🟣 **Lingots d'Empirium** — pour forger la panoplie
* 🎁 **Lot climax au palier 50** — la plus grosse récompense du pass

{% hint style="info" %}
Pense à **réclamer** tes paliers : les récompenses ne sont pas données automatiquement, tu dois cliquer dessus dans `/bp`.
{% endhint %}

{% hint style="info" %}
Complète les **quêtes** du Battle Pass pour gagner de l'XP de pass plus vite et monter les paliers.
{% endhint %}
