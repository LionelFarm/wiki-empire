# 📜 Commandes joueurs

La liste de toutes les commandes disponibles pour les joueurs d'EMPIRE Survival.

{% hint style="info" %}
Ces commandes sont accessibles à **tous les joueurs**. Les commandes staff/admin ne sont pas listées ici.
{% endhint %}

## 🏠 Téléportation

| Commande | Description |
|----------|-------------|
| `/spawn` | Retourner au spawn |
| `/home <nomDuHome>` | Se téléporter au home du même nom |
| `/sethome <nomDuHome>` | Créer un home à ta position actuelle |
| `/rtp` | Se téléporter aléatoirement dans le monde |

## ⚔️ RPG / Personnage

| Commande | Description |
|----------|-------------|
| `/bp` | Ouvrir le Battle Pass |
| `/skills` | Gérer tes sorts actifs |
| `/skilltrees` | Dépenser tes points dans l'arbre de compétences |
| `/relique` | Ouvrir la Forge Mystique (craft de la Relique Vierge) |
| `/forgemaudite` | Ouvrir la Forge Maudite (craft d'armes maudites) |

## 💰 Économie

| Commande | Description |
|----------|-------------|
| `/sell` | Ouvrir le menu de vente au Bourseman |
| `/ah` | Ouvrir l'hôtel de vente entre joueurs |
| `/ah sell <prix>` | Mettre l'objet en main en vente à ce prix |
| `/pb` | Voir ton solde de Points Boutique |
| `/vote` | Voir les sites de vote et les récompenses |

## 💬 Social & Faction

| Commande | Description |
|----------|-------------|
| `/msg <pseudo>` | Envoyer un message privé à un joueur |
| `/r` | Répondre au dernier message privé reçu |
| `/k` | Ouvrir le menu principal de faction (Kingdoms) |
| `/k create <nomFaction>` | Créer une nouvelle faction |
| `/k list` | Afficher les informations sur ta faction |
