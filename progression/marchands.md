# 🛒 Marchands & vente

Le commerce sur EMPIRE Survival passe par des **PNJ marchands** situés au spawn. Approche-les et fais un **clic droit** pour ouvrir leur menu.

## 🧳 Marchand Itinérant

Un marchand au **stock qui tourne chaque jour** :

* **4 objets communs + 2 objets rares** proposés par jour
* Cooldown de **24 h par joueur**
* Le stock change tous les jours : reviens régulièrement !

## 🏛️ Intendant du Village

Le PNJ qui gère ton **niveau de contribution**. Paie les paliers pour débloquer de meilleurs objets.

```
/intendant
```
👉 Voir [Niveau de contribution](contribution.md).

## 💰 Bourseman (vente au serveur)

Le **Bourseman** te permet de **vendre tes ressources** directement au serveur contre des **$**.

* **Clic droit** sur le PNJ → un menu de vente s'ouvre.
* Dépose tes objets pour les vendre instantanément.

{% hint style="info" %}
C'est le moyen le plus simple de transformer tes ressources (minerais, butin de farm…) en argent.
{% endhint %}

## 🏪 Hôtel de vente (entre joueurs)

Pour vendre tes objets **aux autres joueurs** (et non au serveur) :

```
/ah
```

Tu fixes ton prix, les autres achètent, tu touches l'argent. Idéal pour les objets rares (reliques, panoplie Empirium, clés…).
