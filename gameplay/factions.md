# ⚔️ Factions

Crée ou rejoins une **faction** pour t'allier, protéger ton territoire (claims) et faire la guerre. Système géré par **Improved Factions**.

## Commandes essentielles

| Commande | Description |
|----------|-------------|
| `/f create <nom>` | Créer ta faction |
| `/f invite <joueur>` | Inviter un joueur |
| `/f join <faction>` | Rejoindre une faction |
| `/f leave` | Quitter ta faction |
| `/f info` | Infos sur ta faction |
| `/f list` | Liste des factions |
| `/f help` | Aide complète |

## Territoire (claims)

| Commande | Description |
|----------|-------------|
| `/f claim` | Revendiquer le chunk où tu te trouves |
| `/f unclaim` | Libérer le chunk |
| `/f map` | Voir la carte des claims autour de toi |

{% hint style="info" %}
Un chunk **claim** est protégé : seuls les membres de ta faction peuvent y construire et casser des blocs.
{% endhint %}

## Base de faction

| Commande | Description |
|----------|-------------|
| `/f sethome` | Définir la base de la faction |
| `/f home` | Te téléporter à la base de faction |

## Relations & guerre

| Commande | Description |
|----------|-------------|
| `/f ally <faction>` | Proposer une alliance |
| `/f enemy <faction>` | Déclarer un ennemi |

{% hint style="warning" %}
Les factions activent le **power & raids** : si la puissance de ta faction baisse (membres tués), tes claims deviennent vulnérables au pillage. Reste fort !
{% endhint %}

## Tag dans le chat

Quand tu es dans une faction, ton tag apparaît automatiquement dans le chat à côté de ton pseudo.
