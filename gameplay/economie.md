# 💰 Économie & argent

La monnaie du serveur est le **$** (dollar), gérée par **EssentialsX** + **Vault**.

## Commandes de base

| Commande | Description |
|----------|-------------|
| `/bal` | Voir ton solde |
| `/bal <joueur>` | Voir le solde d'un joueur |
| `/pay <joueur> <montant>` | Envoyer de l'argent à un joueur |
| `/baltop` | Classement des plus riches |

## Comment gagner de l'argent ?

* ⚔️ **Tuer des joueurs sous prime** → voir [Primes](primes.md)
* 🛒 **Vendre tes ressources** au **Bourseman** (PNJ vendeur) → voir [Marchands & vente](../progression/marchands.md)
* 🎁 **Battle Pass** : chaque palier donne de l'argent → voir [Battle Pass](../progression/battle-pass.md)
* 🏪 **Hôtel de vente** : vends tes objets aux autres joueurs

## L'hôtel de vente (entre joueurs)

Un **hôtel de vente** permet d'acheter et vendre des objets entre joueurs :
```
/ah
```

{% hint style="info" %}
Tu mets un objet en vente avec ton prix ; les autres joueurs peuvent l'acheter. Tu récupères l'argent automatiquement.
{% endhint %}

## Dépenser ton argent

* 🛒 **Marchands** (Marchand Itinérant, Intendant) — stock qui tourne chaque jour
* 🎟️ **Niveau de contribution** — monte de palier pour débloquer de meilleurs objets
* 🏪 **Hôtel de vente** — achète aux autres joueurs
