# 🎨 Ton skin en jeu

Le serveur utilise **SkinsRestorer** pour que tu puisses avoir un vrai skin, même en compte cracké.

## Si ton pseudo = ton compte Mojang/Java

Ton skin s'applique **automatiquement** dès ta première connexion. Rien à faire ! 🎉

## Si tu joues avec un pseudo inventé (cracké)

Tape une seule fois :
```
/skin <pseudo_mojang>
```

**Exemple :** `/skin Notch` pour récupérer le skin de Notch.

## Changer de skin

Tu peux changer de skin quand tu veux, en reprenant la commande avec un autre pseudo :
```
/skin <autre_pseudo>
```

| Commande | Description |
|----------|-------------|
| `/skin <pseudo>` | Appliquer le skin d'un compte Minecraft |
| `/skin clear` | Revenir à ton skin par défaut |
| `/skin update` | Forcer la mise à jour de ton skin |

{% hint style="info" %}
Pourquoi cette commande ? En offline mode, Minecraft ne contacte pas Mojang pour récupérer les skins. SkinsRestorer le fait à sa place.
{% endhint %}
