# 🏰 Royaumes (Kingdoms)

Les **Royaumes** permettent de créer des communautés organisées avec des territoires protégés, des alliés et des ennemis. Rejoins ou crée ton royaume pour dominer le monde d'EMPIRE Survival.

---

## 🚀 Créer un Royaume

```
/k create <nom>
```

Un **Nexus** sera automatiquement placé à ta position — c'est le cœur de ton royaume. Protège-le !

---

## 🗺️ Claims — Protéger son territoire

| Commande | Description |
|----------|-------------|
| `/k claim` | Revendiquer le chunk où tu te trouves |
| `/k unclaim` | Libérer un chunk |
| `/k map` | Afficher la carte des territoires |
| `/k visualize` | Visualiser les limites de tes claims |

{% hint style="info" %}
Un **chunk** est une zone de 16×16 blocs. Les chunks claimés sont **protégés** — personne d'autre ne peut y construire ou détruire.
{% endhint %}

## 👥 Gérer son Royaume

| Commande | Description |
|----------|-------------|
| `/k invite <joueur>` | Inviter un joueur |
| `/k kick <joueur>` | Expulser un membre |
| `/k leave` | Quitter son royaume |
| `/k disband` | Dissoudre le royaume |
| `/k king <joueur>` | Transférer le titre de roi |
| `/k members` | Voir la liste des membres |
| `/k show` | Afficher les infos du royaume |
| `/k lore <texte>` | Définir la description du royaume |
| `/k rename <nom>` | Renommer le royaume |
| `/k tag <tag>` | Définir le tag du royaume |

## 🏠 Base du Royaume

| Commande | Description |
|----------|-------------|
| `/k sethome` | Définir la base du royaume |
| `/k home` | Se téléporter à la base |
| `/k unsethome` | Supprimer la base |
| `/k nexus` | Aller au Nexus |

---

## 💰 Banque du Royaume

| Commande | Description |
|----------|-------------|
| `/k bank deposit <montant>` | Déposer des coins dans la banque |
| `/k bank withdraw <montant>` | Retirer des coins |

{% hint style="warning" %}
La banque sert à payer les taxes et l'entretien du royaume. Gardez-la alimentée !
{% endhint %}

## ⚠️ Règles importantes

- Le **Nexus** est la structure vitale de ton royaume — s'il est détruit lors d'une guerre, ton royaume est vulnérable
- Les chunks non claimés (**Wilderness**) sont ouverts à tous — construis dans ta zone claimée pour être protégé
- Les **taxes** sont prélevées automatiquement de la banque du royaume
